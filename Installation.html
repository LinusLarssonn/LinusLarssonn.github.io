Guide routerinstallation

1. Koppla in din router med eladaptern i ett eluttag.

2. Koppla in ethernetkabeln/internetkabeln från antingen modem eller fiberuttag.

3. Starta routern om den inte gör det automatiskt.

4. Starta din dator som du ska använda det trådlösa nätverket med och stoppa in CD-skivan som följde med routern.

5. Följ instruktionerna i programmet och se till att välja WPA2-säkerhet. Se hur du skapar ett säkert lösenord via denna länk: McAfee

6. Anslut till nätverket via Wi-Fi på din dator.
